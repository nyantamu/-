ダミーカーソル実験を再現してみました。
Python 3.1.1で動作していることを確認しています。

# 動作説明
1. 実行すると画面中央に実験画面が表示され、3カウントの後に実験が開始されます。

2. 本物と偽物のカーソルが画面上に現れます。カーソルは本物と偽物が全て赤い円で表示され、それぞれに番号が振られています。

3. カーソルの番号を指摘し、指摘したカーソルが本物か偽物かを判定します。指摘の際にはキーボードで数値を入力し、Enterを押してください。間違えてもBackspaceで消すことができます。

4. 結果はcsvファイルで出力されます。

5. 終了したいときは、キーボードでendを入力すると終了できます。また、規定回数を超えると自動的に終了します。
